# Vigihdev DTO

`vigihdev/dto` adalah library **Data Transfer Object (DTO)** untuk ekosistem Vigihdev.  
Dibuat dengan tujuan agar proses **serialisasi, deserialisasi, dan validasi data** menjadi lebih aman, eksplisit, dan mudah diuji.
